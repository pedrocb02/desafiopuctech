Projeto colaborativo desenvolvido em grupo usando Git e GitHub. O tema é uma Pokédex em Python com operações e funcionalidades extras.

Como rodar:
1. Clone o repositório
 git clone https://github.com/pedrocb02/desafiopuctech.git

2. Entre na pasta do projeto:
 cd pokedex

3. rode o projeto:
 python .\pokedex.py

Funcionalidades
 - Adicionar, listar, atualizar e remover pokémons
 - Buscar pokémon por nome ou tipo
 - Listar pokémons ordenados e filtrados por nível
 - Validação de tipos aceitos e níveis
 - Limpar a pokédex
